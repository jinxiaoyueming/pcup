Classpublic class Dbg
InheritanceDbg Object

Dbg 类用于在舞台上显示调试信息。

在PC上可以有以下键盘控制：

  • 空格键：切换调试信息的visible属性。
  • V键：切换调试信息的mouseEnabled属性。
  • X键：清空调试信息。

    • 在移动设备上没有键盘控制，就在初始化时禁用鼠标事件（相当于PC上V键的功能）。这样可以在显示调试信息的情况下操作程序。

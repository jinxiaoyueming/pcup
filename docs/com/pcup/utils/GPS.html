Packagecom.pcup.utils
Classpublic class GPS
InheritanceGPS Object

GPS类有两个功能：
  • 可获取设备当前位置的经纬度。
  • 计算两点间的距离（单位：千米）。

    position:Point  [read-only]

    当前位置的经纬度（x为经度，y为纬度）。

    以下四种情况会返回 null 值：

  • 当前设备不支持传感器。
  • 程序被禁止使用传感器（例如发布Android版本时没有在XML配置中添加GPS权限）。

  • 未获取到GPS数据。

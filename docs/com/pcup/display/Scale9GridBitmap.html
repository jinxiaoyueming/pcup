Classpublic class Scale9GridBitmap
InheritanceScale9GridBitmap flash.display.Bitmap

Scale9GridBitmap 类通过对 Bitmap 类的 width、height、scale9Grid 三个属性的重写来完成位图的九宫格缩放。

[声明]原作者为：itamt[at]qq.com。源码地址为：http://code.google.com/p/tcodes/source/browse/trunk/src/msc/display/Scale9GridBitmap.as

PH修改：

  • 1、修复当设置 width、height 值小于1时会出错的bug。办法：在draw()中过滤掉小于1的情况.

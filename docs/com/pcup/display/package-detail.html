com.pcup.displayClasses
 


Classes
 ClassDescription
 Button Button 类的实例是一个按钮。由传入的两个素材来实现按钮的普通和激活两个状态。
 CheckBox 复选框。 可通过 CheckBox.select 来获取选择状态
 ColorPicker ColorPicker 类的实例是一个颜色选择器。 TODO：设置当前颜色值还只能改变色块的颜色，三个颜色轴的值没有同步。
 DragSelect DragSelect 类的实例是一个可拖动的选择集。 传入一个数组作为此选择集的数据来源，拖动选择后会调度 flash.events.Event.SELECT 事件，以便获取当前选择项。
 DragSpr DragSpr 类的实例是一个可拖动的 Sprite 对象。 会自动为拖动对象生成遮罩。遮罩在手动方向上的长度由构造方法中的 L 参数来确定，另一方向上的长度会根据添加的对象的大小自动调整。 说明：所有被添加到拖动容器（subCtn）中的对象我们称之为“子元素”。
 PageDot PageDot 类的实例是作为 PageSpr 对象的配置项而存在，这就是页标记点。
 PageSpr PageSpr 类的实例是一个可拖动翻页的 Sprite 对象。 会自动为拖动对象生成遮罩。遮罩在手动方向上的长度由构造方法中的 L 参数来确定，另一方向上的长度会根据添加的对象的大小自动调整。 说明：所有被添加到拖动容器（subCtn）中的对象我们称之为“子元素”。
 PictureWheel Wheel effects of picture browsing.
 Radio 单选按钮组 可通过 Radio.selectIndex 来获取或设置选择项
 ScrollBar ScrollBar 类的实例是作为 DragSpr 对象的配置项而存在，这就是滚动条。 缺省的滚动条素材是用代码画的灰色方块，可通过 setMaterail() 方法来更新素材。
 Twotouch The instance of Twotouch that can be drag/zoom/rotate.
 ValueBar ValueBar 类的实例是一个取值条。如同多媒体播放器中的声音调整条。

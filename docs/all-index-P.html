PageSpr — class, package com.pcup.display
PageSpr 类的实例是一个可拖动翻页的 Sprite 对象。 会自动为拖动对象生成遮罩。遮罩在手动方向上的长度由构造方法中的 L 参数来确定，另一方向上的长度会根据添加的对象的大小自动调整。 说明：所有被添加到拖动容器（subCtn）中的对象我们称之为“子元素”。
